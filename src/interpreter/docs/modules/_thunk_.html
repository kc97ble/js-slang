  • getContextWithIndependentEnvironment(context: Context): Context
    • Defined in home/spring/Documents/js-slang/src/interpreter/thunk.ts:13

    Returns a copy of context where context.runtime.environments is shallow-copied.

    In lazy-evaluation mode, delayed evaluations are stored as a Thunk. A Thunk contains a node and a context. Because context is mutable, a mutation to the context outside a Thunk can mutate the context inside the Thunk. This function is created to resolve that issue.

isThunkAware

  • isThunkAware(fun: Value): boolean
    • Defined in home/spring/Documents/js-slang/src/interpreter/thunk.ts:97

    Checks whether fun is thunk-aware.

    A function is thunk-aware if it can accept a Thunk as an argument. Most of built-in functions are not thunk-aware. To distinguish thunk-aware functions from non-thunk-aware functions, all thunk-aware functions are required to have the property isThunkAware to be set to true.

    Parameters

    • fun: Value

      the function to be checked

    Returns boolean

    true if fun is thunk-aware, false otherwise
